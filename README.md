# RexSys

**RexSys** adalah sistem manajemen stok sederhana yang digunakan oleh perusahaan konveksi *Rexsy Collection* (produsen busana bayi).  
Aplikasi ini membantu dalam mendata dan memantau pergerakan stok barang masuk dan keluar secara efisien.

---

## Fitur Utama
- **Login Sistem** — Akses aman dengan autentikasi pengguna.
- **Dashboard Ringkasan** — Menampilkan total barang masuk, keluar, dan stok akhir per kategori.
- **Manajemen Kategori Barang** — CRUD kategori barang.  
- **Manajemen Barang** — CRUD data barang dengan satuan (lusin, seri, dll).  
- **Barang Masuk** — Pencatatan stok masuk berdasarkan tanggal dan kategori.  
- **Barang Keluar** — Pencatatan stok keluar secara rinci.  

---

## Teknologi yang Digunakan
- **Bahasa Pemrograman:** PHP Native  
- **Database:** MariaDB  
- **Frontend:** HTML, CSS, Bootstrap, DataTables, jQuery

---

## Tampilan Aplikasi
| Halaman | Deskripsi |
|----------|------------|
| Login | Form autentikasi pengguna sebelum mengakses sistem |
| Dashboard | Ringkasan data stok & aktivitas |
| Kategori | CRUD kategori barang (misal: Stok Singlet, Stok Tas, dll) |
| Barang | CRUD data barang sesuai kategori |
| Barang Masuk | Pencatatan dan histori stok masuk |
| Barang Keluar | Pencatatan dan histori stok keluar |
